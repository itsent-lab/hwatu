# Hwatu

React와 ASP.NET Core로 만든 한국 화투 웹게임입니다. 어르신과 가족이 쉽고 편안하게 즐길 수 있는 건강한 여가 환경을 목표로 제작했으며, 2인 맞고와 3인 고스톱, AI 상대, 게임 저장 및 사용자 관리 기능을 포함합니다.

## 프로젝트 목적

- 어르신의 접근성과 편의를 고려한 웹 기반 맞고·고스톱 게임
- 가족, 복지관, 비상업적 연구 및 학습 환경에서 무료로 활용할 수 있는 오픈소스 프로젝트
- 낙장불입으로 인한 불편을 줄이는 판당 무르기와 게임머니 리필 등 편의 기능 제공
- 상용 화투 게임에서 익숙한 조작 흐름을 참고하되, 웹 환경에 맞게 화면 구성과 동작을 독립적으로 재설계

## 제작 방식과 자산 출처

이 프로젝트는 특정 상용 게임 클라이언트를 디컴파일하거나 스크래핑하지 않았으며, 해당 클라이언트에 포함된 PNG, WAV 등의 파일을 직접 추출해 사용하지 않았습니다. 화면과 상호작용은 참고 영상을 바탕으로 AI 코딩 및 이미지 생성 도구의 도움을 받아 새로 구현했습니다.

일반적인 화투 게임 규칙, 기능 아이디어, 조작 흐름과 UI 배치 개념을 참고하되 구체적인 소스 코드와 화면 표현은 이 프로젝트를 위해 독립적으로 작성했습니다. 다만 일반적인 아이디어·규칙과 달리 특정 서비스의 고유한 그래픽, 문구, 음원, 코드 및 창작적인 표현은 별도의 권리로 보호될 수 있으므로 이를 그대로 복제하지 않는 것을 원칙으로 합니다.

저장소에 포함된 화투패 SVG는 AI 생성물이 아니라 Wikimedia Commons에서 적법한 라이선스로 제공되는 자산입니다. 상세 출처와 라이선스는 아래 라이선스 절과 자산별 검증 정보를 확인해 주세요.

> 이 설명은 프로젝트의 제작 과정과 공개 범위를 알리기 위한 것으로, 특정 서비스와의 제휴·승인 관계를 의미하거나 개별 사안에 대한 법률 의견을 제공하지 않습니다.

## 주요 기능

- 2인 맞고와 3인 고스톱 게임 모드
- 난이도별 AI와 게임 진행 효과
- 판당 무르기와 게임머니 리필 등 접근성·편의 기능
- 사용자 로그인, 관리자 계정 관리, 프로필 이미지
- MySQL 기반 게임 저장과 정산 기록
- 반응형 UI 및 PWA 정적 자산
- Vitest 기반 규칙·UI·시뮬레이션 테스트

## 기술 스택

- React 19, TypeScript, Vite
- ASP.NET Core 10, Dapper, MySqlConnector
- MySQL 8 이상
- Vitest

## 로컬 실행

Node.js, .NET 10 SDK, MySQL을 준비합니다.

```bash
npm install
```

프로젝트 루트에 Git에서 제외되는 `appsettings.Local.json`을 만들고 로컬 DB 연결 문자열을 입력합니다.

```json
{
  "ConnectionStrings": {
    "Hwatu": "Server=127.0.0.1;Port=3306;Database=hwatu;User ID=hwatu;Password=change-me;Character Set=utf8mb4;"
  }
}
```

DB와 사용자를 만든 뒤 서버와 프런트엔드 개발 서버를 각각 실행합니다. 서버 시작 시 `database/schema.sql`을 적용합니다.

```bash
npm run dev:server
npm run dev
```

브라우저에서 `http://127.0.0.1:5234`에 접속하면 됩니다. 최초 실행 시 관리자 계정을 등록할 수 있습니다.
