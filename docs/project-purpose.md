# 프로젝트 목적과 공개 원칙

## 1. 프로젝트 목적

화투는 순전히 가족이 함께 사용하기 위해 만드는 가족용 화투 게임 프로젝트입니다.

- 가족이 익숙하고 편안하게 맞고와 고스톱을 즐길 수 있는 환경을 만드는 것이 최우선 목적입니다.
- 불특정 다수를 대상으로 사용자를 모집하거나 공개 서비스를 운영하는 것을 목적으로 하지 않습니다.
- 상업적 서비스, 수익화, 대규모 사용자 확보 또는 범용 게임 플랫폼으로의 확장을 목표로 하지 않습니다.
- 기능과 운영 방식은 일반 대중의 요구보다 실제 가족 사용자의 편의와 필요를 우선합니다.

## 2. 서비스 운영 범위

- 이 프로젝트로 만든 앱은 가족을 위한 제한된 환경에서 사용합니다.
- 인터넷에 소스 코드를 공개하는 것과 불특정 다수에게 서비스를 공개하는 것은 서로 다른 일입니다.
- 코드가 공개되어 있더라도 공식 공개 서비스의 제공, 지속적인 운영, 사용자 지원 또는 서비스 수준을 약속하는 것은 아닙니다.
- 가족 외 사용자를 위한 공개 회원 모집이나 일반 공개 운영을 전제로 개발하지 않습니다.
- 가족용·비상업·비공개 운영은 제3자의 프로그램 내용이나 창작물을 허가 없이 가져올 수 있는 근거가 아니며, 공개 여부와 관계없이 각 권리와 라이선스를 존중합니다.

## 3. 코드를 공개하는 이유

코드를 공개하는 목적은 다른 사람이 이 프로젝트를 가져가 자유롭게 사용하고 수정하는 데 불필요한 제약을 두지 않기 위해서입니다.

- 누구든지 코드를 살펴보고 자기 환경에 맞게 사용하거나 수정할 수 있도록 공개합니다.
- 재사용, 수정, 배포 및 파생 작업을 인위적으로 제한하려는 목적이 없습니다.
- 코드 공개는 화투 자체를 공개 서비스로 운영한다는 의미가 아닙니다.
- 공개 저장소의 존재만으로 원 개발자가 설치, 운영, 유지보수 또는 기술 지원을 제공할 의무가 생기지는 않습니다.
- GitHub에는 개발 소스 코드, 테스트, 문서와 재배포 가능한 자산만 공개하며, 실행 중인 서버, 운영 계정·데이터, 실제 운영 설정이나 서비스 이용 권한은 제공하지 않습니다.
