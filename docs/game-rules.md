# 맞고·고스톱 게임 규칙

이 문서는 FamilyHwatu 앱에 실제로 구현된 맞고와 고스톱 규칙을 설명합니다. 지역별·서비스별 화투 관습이 아니라 현재 코드의 동작을 기준으로 하며, 일반적인 규칙과 구현이 다르면 이 문서에 적힌 현재 구현을 우선합니다.

- 기준일: 2026-07-21
- 맞고: 2인 게임
- 고스톱: 3인 게임
- 게임머니 단위: 냥
- 게임머니는 현금이 아니며 환전할 수 없습니다.

## 1. 공통 구성

### 1.1 패 구성

- 기본 화투패는 1월부터 12월까지 월별 4장씩, 총 48장입니다.
- 두 게임 모두 기본 48장에 보너스패 2장을 더한 50장을 사용합니다.
- 보너스 쌍피는 피 2장, 보너스 삼피는 피 3장으로 계산합니다.

### 1.2 한 차례의 기본 진행

1. 자기 손패에서 1장을 냅니다.
2. 같은 월의 바닥패가 있으면 해당 패와 짝을 맞춰 획득합니다.
3. 더미에서 1장을 뒤집습니다.
4. 뒤집은 패도 같은 방식으로 바닥패와 짝을 맞추거나 바닥에 놓습니다.
5. 특수 상황과 점수를 판정한 뒤 다음 사람에게 차례가 넘어갑니다.

같은 월의 바닥패 수에 따른 처리는 다음과 같습니다.

| 같은 월의 바닥패 | 처리 |
| --- | --- |
| 0장 | 낸 패 또는 뒤집은 패를 바닥에 놓습니다. |
| 1장 | 해당 바닥패와 새 패를 함께 획득합니다. |
| 2장 | 둘 중 1장을 선택해 새 패와 함께 획득합니다. |
| 3장 | 바닥의 3장과 새 패를 모두 획득합니다. |

### 1.3 공통 점수표

맞고와 고스톱은 같은 패 점수 체계를 사용합니다.

| 분류 | 조건 | 점수 |
| --- | --- | ---: |
| 비삼광 | 비광을 포함한 광 3장 | 2점 |
| 삼광 | 비광 없이 광 3장 | 3점 |
| 사광 | 광 4장 | 4점 |
| 오광 | 광 5장 | 15점 |
| 열끗 | 열끗 5장부터 | 1점, 이후 1장마다 +1점 |
| 고도리 | 2월·4월·8월의 새 패 3장 | 5점 |
| 띠 | 띠 5장부터 | 1점, 이후 1장마다 +1점 |
| 홍단 | 1월·2월·3월 홍단 | 3점 |
| 청단 | 6월·9월·10월 청단 | 3점 |
| 초단 | 4월·5월·7월 초단 | 3점 |
| 피 | 피 10장부터 | 1점, 이후 1장마다 +1점 |

- 열끗 점수와 고도리 점수는 함께 더합니다.
- 띠 점수와 홍단·청단·초단 점수는 함께 더합니다.
- 쌍피는 피 2장, 삼피는 피 3장으로 셉니다.
- 9월 국진은 획득한 사람이 열끗 1장 또는 쌍피 1장 중 하나로 선택합니다. 쌍피로 바꾸면 열끗 수에서는 제외합니다.

### 1.4 보너스패

- 손에 든 보너스패를 내면 즉시 획득하고 더미에서 대체 패 1장을 받아 같은 차례를 계속합니다.
- 뒤집기에서 보너스패가 나오면 즉시 획득하고 일반 패가 나올 때까지 계속 뒤집습니다.
- 보너스패로 피를 가져올 때는 일반 피처럼 가치가 낮은 패를 먼저 가져오며, 일반 피가 없으면 쌍피나 삼피 1장을 통째로 가져옵니다.
- 피를 가진 상대가 없으면 가져오는 피 없이 보너스패만 획득합니다.

### 1.5 점당 금액

두 게임 모두 다음 점당 금액을 지원합니다.

- 100냥
- 1,000냥
- 2,000냥
- 5,000냥
- 10,000냥

## 2. 맞고 규칙

### 2.1 인원과 패 나누기

- 사람 1명과 컴퓨터 1명이 진행합니다.
- 각자 손패 10장을 받고 바닥에 일반 패 8장을 놓습니다.
- 바닥에 보너스패가 나오면 선이 가져가며, 바닥의 일반 패가 8장이 될 때까지 더미에서 보충합니다.
- 남은 패는 뒤집기 더미로 사용합니다.

### 2.2 선 정하기

- 첫 판에는 1월부터 12월까지 월별 1장씩 제시된 패에서 사람과 컴퓨터가 각각 1장을 고릅니다.
- 오전 6시부터 오후 6시 전까지는 높은 월이 선이고, 그 밖의 시간에는 낮은 월이 선입니다.
- 같은 월이면 `광 > 열끗 > 띠 > 쌍피 > 피` 순서로 선을 정합니다.
- 월과 패 등급까지 같으면 선이 정해지지 않은 동률입니다.
- 승부가 난 다음 판에는 직전 판의 승자가 선을 이어갑니다.

### 2.3 고·스톱

- 획득 패 점수가 7점 이상이면 고 또는 스톱을 선택할 수 있습니다.
- 이미 고를 선언했다면 직전 고 선언 때보다 획득 패 점수가 높아져야 다시 선택할 수 있습니다.
- 스톱을 선택하면 즉시 판을 끝내고 최종 점수를 계산합니다.
- 고를 선택하면 고 횟수와 당시 점수를 기록하고 상대 차례로 진행합니다.
- 양쪽이 손패와 폭탄 후 남은 뒤집기 차례를 모두 사용했을 때 현재 행동한 사람이 7점 이상이면 자동 스톱합니다.
- 모든 차례가 끝났는데 7점에 도달하지 못하면 나가리입니다.

고 점수와 배수는 다음과 같습니다.

| 고 횟수 | 계산 |
| ---: | --- |
| 0고 | 획득 패 기본 점수 |
| 1고 | 기본 점수 + 1점 |
| 2고 | 기본 점수 + 2점 |
| 3고 | `(기본 점수 + 3점) × 2` |
| 4고 | `(기본 점수 + 4점) × 4` |
| 5고 이상 | 고가 하나 늘 때마다 앞 단계의 고 배수가 2배로 증가 |

### 2.4 뻑과 피 가져오기

| 상황 | 처리 |
| --- | --- |
| 뻑 | 바닥에 같은 월 1장이 있을 때 낸 패와 뒤집은 패가 모두 같은 월이면 3장을 바닥에 남깁니다. |
| 싼 패 먹기 | 상대가 만든 뻑의 3장을 먹으면 상대 피 1장을 추가로 가져옵니다. |
| 자뻑 | 자기가 만든 뻑의 3장을 다시 먹으면 상대 피 2장을 추가로 가져옵니다. |
| 3뻑 | 한 사람이 세 번째 뻑을 만들면 즉시 기본 7점으로 승리합니다. |

- 3뻑 승리에는 고, 흔들기·폭탄, 미션 및 박 배수를 적용하지 않습니다.
- 나가리 이월 배율이 있는 판에서는 3뻑 기본 7점에도 이월 배율을 적용합니다.

### 2.5 쪽·따닥·싹쓸이

| 규칙 | 조건 | 추가 효과 |
| --- | --- | --- |
| 쪽 | 같은 월의 바닥패가 없는 곳에 패를 냈고 뒤집은 패가 그 패와 같은 월일 때 | 상대 피 1장을 가져옵니다. |
| 따닥 | 같은 월의 바닥패 2장 중 1장을 낸 패로 먹고, 뒤집은 패로 나머지 1장을 먹을 때 | 상대 피 1장을 가져옵니다. |
| 싹쓸이 | 한 차례의 획득 처리가 끝난 뒤 바닥패가 0장이 될 때 | 상대 피 1장을 가져옵니다. |

- 한 차례에 여러 조건을 충족하면 각 효과를 함께 적용합니다.
- 마지막 손패를 내는 차례에는 쪽·따닥·싹쓸이의 추가 피 가져오기를 적용하지 않습니다.

### 2.6 흔들기와 폭탄

#### 흔들기

- 손에 같은 월 패 3장이 있고 바닥에 그 월 패가 없으면 흔들기를 선언할 수 있습니다.
- 흔들기를 선언하면 3장을 공개한 뒤 그중 1장을 내야 합니다.
- 같은 월로는 한 번만 흔들 수 있습니다.
- 흔들기 1회마다 최종 정산의 흔들기·폭탄 배수가 2배가 됩니다.

#### 폭탄

- 손의 같은 월 패 3장과 바닥의 같은 월 패 1장을 한꺼번에 먹는 일반 폭탄을 사용할 수 있습니다.
- 손의 같은 월 패 2장과 바닥의 같은 월 패 2장을 한꺼번에 먹는 두장폭탄도 사용할 수 있습니다.
- 폭탄을 사용하면 상대 피 1장을 가져옵니다.
- 폭탄 1회는 흔들기 1회와 같은 2배 배수로 계산합니다.
- 한꺼번에 낸 손패 수에서 1을 뺀 만큼 이후 자기 차례에는 손패를 내지 않고 더미 패만 뒤집습니다. 일반 폭탄은 2회, 두장폭탄은 1회가 이에 해당합니다.
- 폭탄 처리 후 싹쓸이 등 다른 특수 조건을 충족하면 추가 피 가져오기를 함께 적용합니다.

흔들기와 폭탄의 누적 배수는 `2^(흔들기 횟수 + 폭탄 횟수)`입니다.

### 2.7 총통

- 처음 나눈 바닥패에 같은 월 4장이 모두 있으면 즉시 나가리입니다.
- 사람과 컴퓨터가 각각 손에 총통을 동시에 가지고 있으면 즉시 나가리입니다.
- 한쪽만 손에 같은 월 4장을 가지고 있으면 총통 보유자가 다음 중 하나를 선택합니다.
  - 스톱: 기본 7점으로 즉시 승리합니다. 고·흔들기·박 배수는 붙지 않습니다.
  - 계속: 총통을 공개하고 판을 계속하며 4장 흔들기 1회를 정산 배수에 반영합니다.
- 나가리 이월 배율이 있는 판에서는 총통 스톱의 기본 7점에도 이월 배율을 적용합니다.

### 2.8 각패 미션

- 매 판 일반 화투패 중 서로 다른 3장을 미션패로 정합니다.
- 승자는 자기가 획득한 미션패 수에 따라 다음 배수를 받습니다.

| 획득한 미션패 | 미션 배수 |
| ---: | ---: |
| 0장 | ×1 |
| 1장 | ×2 |
| 2장 | ×4 |
| 3장 | ×8 |

### 2.9 박

각 박은 2배이며 여러 박이 동시에 성립하면 서로 곱합니다.

| 박 | 성립 조건 |
| --- | --- |
| 피박 | 승자가 피 10장 이상으로 피 점수를 냈고 패자가 가진 피가 7장 이하일 때 |
| 광박 | 승자가 오광을 완성했거나, 승자가 광 3장 이상이고 패자에게 광이 없을 때 |
| 멍박 | 패자에게 열끗이 한 장도 없을 때 |
| 고박 | 패자가 한 번 이상 고를 선언한 뒤 점수를 더 올리지 못한 상태에서 상대가 스톱할 때 |

예를 들어 피박·광박·멍박이 동시에 성립하면 박 배수는 `2 × 2 × 2 = 8배`입니다.

### 2.10 나가리와 최종 정산

- 나가리 다음 판은 2배입니다.
- 나가리가 연속되면 2배, 4배, 8배처럼 계속 두 배씩 누적합니다.
- 승부가 나면 그다음 판의 이월 배율은 1배로 초기화합니다.

맞고의 최종 점수는 다음 순서로 계산합니다.

1. 획득 패의 기본 점수를 합산합니다.
2. 고 횟수만큼 점수를 더합니다.
3. 3고 이상이면 고 배수를 곱합니다.
4. 흔들기·폭탄 배수를 곱합니다.
5. 미션 배수를 곱합니다.
6. 성립한 박 배수를 모두 곱합니다.
7. 나가리 이월 배수를 곱합니다.

최종 게임머니는 `최종 점수 × 점당 금액`입니다. 맞고에서는 패자 한 명이 승자에게 해당 금액을 지급하며, 실제 이동 금액은 양쪽의 보유 게임머니 한도 안에서 제한됩니다.

## 3. 고스톱 규칙

현재 고스톱 모드는 3인 기본 진행을 구현한 단계입니다. 아래에 적지 않은 전통 규칙은 자동으로 적용되지 않습니다.

### 3.1 인원과 패 나누기

- 사람 1명과 컴퓨터 2명이 진행합니다.
- 각자 손패 7장을 받고 바닥에 일반 패 6장을 놓습니다.
- 바닥에 보너스패가 나오면 사람 플레이어가 가져가며, 바닥의 일반 패가 6장이 될 때까지 더미에서 보충합니다.
- 사람 플레이어가 항상 선입니다.
- 차례는 `사람 → 컴퓨터 A → 컴퓨터 B → 사람` 순서로 반복합니다.

### 3.2 고·스톱

- 획득 패 점수가 3점 이상이면 고 또는 스톱을 선택할 수 있습니다.
- 이미 고를 선언했다면 직전 고 선언 때보다 획득 패 점수가 높아져야 다시 선택할 수 있습니다.
- 고를 선언할 때마다 고 횟수가 1 증가합니다.
- 스톱 점수는 `획득 패 점수 + 고 횟수`입니다.
- 맞고와 달리 3고 이상에 별도의 고 배수를 적용하지 않습니다.
- 모든 사람의 손패가 끝났을 때 마지막 행동자의 획득 패 점수가 3점 이상이면 자동 스톱합니다.
- 모든 손패가 끝났는데 마지막 행동자가 3점에 도달하지 못하면 나가리입니다.

### 3.3 뻑·싹쓸이·보너스패

| 상황 | 처리 |
| --- | --- |
| 뻑 | 맞고와 같이 같은 월 3장을 바닥에 남깁니다. |
| 싼 패 먹기 | 뻑의 3장을 먹으면 나머지 두 상대에게서 피를 각각 1장씩 가져옵니다. |
| 자뻑 | 자기가 만든 뻑을 다시 먹어도 나머지 두 상대에게서 피를 각각 1장씩 가져옵니다. |
| 싹쓸이 | 바닥패를 모두 먹으면 나머지 두 상대에게서 피를 각각 1장씩 가져옵니다. |
| 보너스패 | 보너스패를 얻으면 나머지 두 상대에게서 피를 각각 1장씩 가져옵니다. |

- 상대가 피를 가지고 있지 않으면 그 상대에게서는 가져오지 못합니다.
- 손의 보너스패를 내면 대체 패를 받고 같은 차례를 계속합니다.
- 뒤집기에서 보너스패가 나오면 일반 패가 나올 때까지 계속 뒤집습니다.

### 3.4 현재 적용되지 않는 규칙

현재 고스톱 구현에는 다음 규칙이 적용되지 않습니다.

- 흔들기와 폭탄
- 쪽과 따닥의 추가 피 가져오기
- 총통 판정
- 3뻑 즉시 승리
- 각패 미션
- 피박·광박·멍박·고박
- 3고 이상 별도 배수

### 3.5 나가리와 최종 정산

- 나가리 이월은 맞고와 동일하게 다음 판 2배이며 연속 나가리마다 다시 2배가 됩니다.
- 승부가 나면 그다음 판은 1배로 초기화합니다.
- 최종 점수는 `(획득 패 점수 + 고 횟수) × 나가리 이월 배수`입니다.
- 상대 2명은 각각 `최종 점수 × 점당 금액`을 승자에게 지급합니다.
- 실제 이동 금액은 각 패자의 보유 게임머니와 승자의 최대 보유 한도 안에서 제한됩니다.
- 컴퓨터 플레이어의 게임머니가 0이 되면 다음 진행을 위해 500,000냥으로 충전됩니다.

## 4. 맞고와 고스톱 차이 요약

| 항목 | 맞고 | 고스톱 |
| --- | --- | --- |
| 인원 | 2명 | 3명 |
| 손패 | 각 10장 | 각 7장 |
| 바닥패 | 일반 패 8장 | 일반 패 6장 |
| 선 | 패 선택으로 결정, 이후 직전 승자 | 항상 사람 플레이어 |

| 고 계산 | 고 점수 추가, 3고부터 배수 | 고 횟수만 점수로 추가 |
| 흔들기·폭탄 | 적용 | 현재 미적용 |
| 쪽·따닥 | 적용 | 현재 미적용 |
| 싹쓸이 | 상대 피 1장 | 각 상대 피 1장 |
| 뻑 회수 | 상대 피 1장, 자뻑 2장 | 각 상대 피 1장 |
